# demo2

一个项目库驱动的售前机会分析与跟进辅助 demo。

## 项目简介
这个项目聚焦在售前 / 销售场景中“历史项目经验难以被即时调用”的问题。  
系统会将客户需求描述或拜访纪要解析为结构化 opportunity，再联动历史项目案例库，生成相似项目参考和拜访建议，并承接最小跟进与回看。

## 当前版本主链
当前版本主要验证这条主链是否成立：

**客户需求输入 → 机会对象解析 → 相似项目检索 → 建议生成 → 机会记录留存与回看**

## 当前版本包含
- 历史项目台账导入与清洗
- opportunity 解析
- 相似项目检索
- 建议生成
- 最小跟进承接
- 前台演示页与后台治理台

## 当前版本边界
当前版本重点是验证主链闭环，不包括：
- 完整 CRM
- 权限系统
- 多用户体系
- 正式生产部署
- 完整项目管理能力

## 项目结构
- `app/`：页面入口，包含前台演示页和后台治理台
- `pipeline/`：核心流程，包括数据导入、清洗、机会解析、检索和建议生成
- `services/`：AI 能力接口与 provider 封装
- `data/`：案例数据、修正日志和本地数据载体（部分内容已脱敏或未上传）

## 主要页面
- `app/visit_assistant_app.py`：前台演示页
- `app/project_case_workbench.py`：后台治理台

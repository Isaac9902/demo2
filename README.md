# AI Pre-sales Opportunity Assistant Demo

这是一个项目库驱动的售前机会分析与跟进辅助 demo。

## 项目目标
将客户需求描述或拜访纪要解析为结构化 opportunity，
并联动历史项目案例生成相似项目参考和拜访建议。

## 当前版本包含
- 历史项目台账导入与清洗
- opportunity 解析
- 相似项目检索
- 建议生成
- 最小跟进承接
- 前台演示页与后台治理台

## 当前版本边界
当前版本重点是验证主链闭环，
不包括完整 CRM、权限、多用户和正式生产部署。

## 主要页面
- `app/visit_assistant_app.py`：前台演示页
- `app/project_case_workbench.py`：后台治理台

## 说明
本仓库为 demo / 验证版本，部分数据和配置已做脱敏处理。
